# Project Images

This folder contains images for your featured projects.

## Image Guidelines

For the best results, please follow these guidelines when adding your project images:

### Image Specifications:
- **Dimensions**: 1200 x 800 pixels (3:2 aspect ratio)
- **File Format**: PNG or JPG
- **File Size**: Keep under 500KB for optimal loading
- **Quality**: High quality but optimized for web

### Required Images:
1. `chat-app.png` - Real-Time Chat App screenshot
2. `hospital-management.png` - Hospital Management System screenshot
3. `portfolio.png` - Portfolio Website screenshot

### Tips for Great Project Images:
- Take full-page screenshots of your projects
- Show the most interesting/unique features
- Use a clean, professional layout
- Include both desktop and mobile views if possible
- Add subtle mockups or device frames for better presentation

### How to Add Images:
1. Take screenshots of your projects
2. Resize them to 1200 x 800 pixels
3. Optimize them using tools like TinyPNG or ImageOptim
4. Save them in this folder with the exact names listed above
