📸 Add Project Images Guide

Follow this simple guide to add beautiful images to your featured projects

📋 Image Specifications

  • Dimensions: 1200 x 800 pixels (3:2 aspect ratio recommended)
  • Format: PNG or JPG
  • File Size: Keep under 500KB for fast loading
  • Quality: High resolution but web-optimized

🎯 Required Images

Add these 3 images to your portfolio:

  1. chat-app.png - Screenshot of your Real-Time Chat App
  2. hospital-management.png - Screenshot of your Hospital Management System
  3. portfolio.png - Screenshot of your Portfolio Website
⚠️ Important: File names must match exactly (case-sensitive)

🚀 How to Add Images

Step-by-Step Instructions:

  1. Take Screenshots
    • Open your project in a browser
    • Take full-page screenshots
    • Capture the most interesting features
  2. Edit & Resize
    • Use any image editor (Photoshop, GIMP, or online tools)
    • Resize to 1200 x 800 pixels
    • Crop to show the best part of your project
  3. Optimize
    • Use TinyPNG to compress
    • Or use Squoosh for advanced optimization
    • Keep file size under 500KB
  4. Save to Folder
    • Navigate to: D:\Portfolio\public\images\projects\
    • Save your images with the exact names listed above
  5. Refresh Browser
    • Your images will automatically appear!
    • No code changes needed

💡 Pro Tips

  • 🎨 Use consistent styling across all screenshots
  • 📱 Include both desktop and mobile views if space allows
  • ✨ Add subtle mockups or device frames for professional look
  • 🔍 Ensure text in screenshots is readable
  • 🎭 Show your project's unique features
  • 🌈 Use high-quality, well-lit screenshots
